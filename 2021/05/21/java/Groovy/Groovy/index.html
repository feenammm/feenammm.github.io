Groovy

Groovy是用于Java虛拟机的一种敏捷的动态语言，它是一种成熟的面向对象编程语言，既可以用于面向对象编程，又可以用作纯粹的脚本语言。使用该种语言不必编写过多的代码，同时又具有闭包和动态语言中的其他特性。

特性

  1. Groovy完全兼容java

  2. 分号是可选的

  3. 类/方法默认是public修饰

  4. 编译器会自动给属性添加getter/setter方法

  5. 属性可以通过.直接获取

  6. 最后一个表达式的值会作为返回值

  7. ==相当于java中的equals方法

  8. 没有NullPointerException

  9. 自带assert语句

    1. assert version == 2

  10. 类型是可选的,会进行推导

    1. def version = 1

  11. 方法括号是可选的

    1. println(version)
    2. println version

  12. 字符串有三种表达形式

    1. def s1 = ‘abc’ //字符串

    2. def s2 = ‘abc ${version}’ //可以添加变量

    3. def s3 = ‘’’def

      ​ abc

      ​ ahfd ‘’’ //可以换行

  13. 集合API

    1. list (java中的ArrayList)
      1. 定义
        1. def list = [‘abc’,’def’]
      2. 添加元素
        1. list << ‘ghi’
      3. 长度
        1. list.size()
    2. map (java中的LinkedHashMap)
      1. 定义
        1. def map = [‘abc’:123,’def’:345]
      2. 添加
        1. map.adsabd = 1245
      3. 获取
        1. println map.abc
        2. print map[‘def’]

  14. 闭包

    就是一个代码块,和方法一样,可以有参数,可以被赋值给变量,也可以传递给方法,也可以像普通方法一样调用
